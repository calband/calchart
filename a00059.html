Detailed Description

A view used to link the ContinuityEditor to the CalChartDoc.

When the CalChartDoc updates all of its views, it will send updates to this view, and this view will route those updates to the Continuity Editor. Also, when the Continuity Editor changes a continuity, it informs the CalChartDoc through the view.
