POINTS

To specify a point, one of these symbols MUST be used:

NAMEPOINT
SPlocation on the current stuntsheet
Pcurrent location
NPlocation on the next stuntsheet
R1first reference point on the current stuntsheet
R2second reference point
R3third reference point

First, note that P changes with every beat, but the others are constant throughout a single stuntsheet.

Second, points differ from numerical expressions in that they cannot be combined by arithmetic operators, and cannot be used when a numeric parameter is expected.
