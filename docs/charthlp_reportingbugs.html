About Reporting Bugs

File issues and feature requests to:

https://github.com/calband/calchart/issues/new
Please try to include helpful information to track down the bug. Try to reproduce the bug, and give instructions on how to duplicate it, or include a show that illustrates the problem. And Include the date from the About window so we know what version you're using. Comments or suggestions are also appreciated.

Known issues are listed at:

https://github.com/calband/calchart/issues
