Windows Installation

You need an unzip program installed. Unzip the software with the command:

unzip calchart.zip
or 
pkunzip -d calchart.zip
Easy Installation

For simple installation, choose Run from the Program Manager's menu. Run the install.exe program you extracted. Enter the path you want and click on Okay. A group called Cal Band will be created with an icon for CalChart.

Printer Configuration

In general, there are two ways of printing, Native Printing, and Post-Script Printing. For windows, Native Printing may have sub-optimal results, so you may want to do Post-Script printing. With Post-Script printing CalChart can print to PostScript printers (such as laser printers). You can also print to a PostScript file and use ghostscript (a free PostScript interpreter) to s print to another printer.
