# Printed Continuity Codes

| **CODE** | **MEANING** |
|----------|-------------|
| `%%`NAME | Begin a sheet titled NAME |
| `\po`    | plainman |
| `\pb`    | backslashman |
| `\ps`    | slashman |
| `\px`    | xman |
| `\so`    | solidplainman |
| `\sb`    | solidbackslashman |
| `\ss`    | solidslashman |
| `\sx`    | solidxman |
| `\bs`    | start boldface |
| `\be`    | end boldface |
| `\is`    | start italics |
| `\ie`    | end italics |
| `<`      | line appears only on continuity sheets |
| `>`      | line appears only on stuntsheets |
| `~`      | center line |

The symbols `<>~` must appear in that order at the start of the line.
The file must begin with `%%`.

Use tabs to align the measure number, symbol, and continuity columns.
