# Obtaining the Software

The latest version of CalChart may be obtained from the official
CalChart web page:

[http://sourceforge.net/projects/calchart](http://sourceforge.net/projects/calchart)

There are currently versions for Windows and Mac.  UNIX and Linux platforms are also possible, but you'll have to compile it yourself.
