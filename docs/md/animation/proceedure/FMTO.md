# FMTO

Format: `FMTO <destination>`

`FMTO` marches in a straight line to the point `<destination>`.  It
infers the stepsize and then determines how many steps to take using
that stepsize.  (See the section on [Step Sizes](../STEPSIZES.md) for
more information on guessing).
