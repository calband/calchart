# MAGIC

Format: `MAGIC <destination>`

`MAGIC` is special: `P` is moved directly to <destination>, using the
*last beat* that was marched.  In other words, it uses up no beats.

`MAGIC` is generally used when animation either is impossible (e.g. Script
Cal) or is undesired.
