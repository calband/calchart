# OPP

Format: `OPP(<direction>)`

`OPP` simply returns a direction opposite to the one specified.

Example:

```
  OPP(DIR(NP))
```

returns the direction which is directly away from `NP`.

See [Directions](../DIRECTIONS.md) for details on the mathematics of this.
