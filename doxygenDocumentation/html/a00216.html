Modifies the input parameters so that they represent a unit vector pointing in a particular direction (with a special case when the direction is a diagonal direction).

When the direction is a diagonal direction, the components of the output vector both will have absolute values equal to one.

Parameters
aThis will be modified by the function to represent the x component of the output vector.
bThis will be modified by the function to represent the y component of the output vector.
dirThis is the angle that the final output vector will make with the positive x-axis, in degrees.

Modifies the input vector so that it has a particular magnitude and direction (except in the case that the direction is diagonal).

When diagonal vectors are formed, the absolute values of both final vector components will have the magnitude passed as a parameter.

Parameters
cThe vector to modify.
dirThe angle that the finished vector will make with the positive x-axis, in degrees.
magThe magnitude of the final vector, except in the case where the direction is diagonal. In that case, both components of the final vector will have absolute values equal to this parameter.
