Detailed Description

The dialog through which the user can select/deselect points.

This dialog appears when the user selects the "Point Selections" option in the edit menu of the field frame.

void PointPicker::PointPickerSelect ( wxCommandEvent &  )

Makes sure that the selection which the user defined in the PointPicker becomes the active selection in the show itself.

This is used to select/deselect individual points. The change in the show's selection will cause the show to update all of its views, which will cause the update() method of this object to be called, thereby making sure that the selection in this object matches the selection in the show.

void PointPicker::Update ( )

Called when the CalChartDoc updates its views.

This method is called by a PointPickerView, and makes sure that the selection in the PointPicker matches the selection in the show itself.
