Detailed Description

A structure containing all data required to add a tool to a toolbar through wxWidgets.

Member Data Documentation

wxBitmap* ToolBarEntry::bm

The image for the tool.

wxString ToolBarEntry::desc

The description of the tool.

This appears when the user hovers his/her mouse over the tool.

int ToolBarEntry::id

The identity of the tool.

This id is attached through wxWidgets to something that can actually perform the behavior of the tool, like a method. This id, therefore, defines the behavior that results from working with this tool.

wxItemKind ToolBarEntry::kind

The tool's type (e.g.

a dropdown menu or toggle button). A list of tool types can be found at: http://docs.wxwidgets.org/trunk/defs_8h.html#abd0c640814a55e4adda3bce698646d2d
