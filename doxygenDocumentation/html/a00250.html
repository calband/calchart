anonymous enum

This enumeration keeps track of the identities that CalChart has registered with wxWidgets.

This comes into play when registering tools, for example: when you register a tool with wxWidgets, you link a method to an integer identity, and if you link an option in a toolbar to the SAME identity as the method, then when you select that option from the toolbar, it will call the method associated with that integer identity to handle the event.
