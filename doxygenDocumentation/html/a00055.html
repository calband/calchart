Detailed Description

Represents the EITHER function in the Continuity Editor.

Syntax in Continuity Editor: EITHER([direction 1] [direction 2] [reference]) This function returns [direction 2] if it is closer to the direction between the current point and [reference], and returns [direction 1] otherwise.
