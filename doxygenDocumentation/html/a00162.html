Detailed Description

A matrix that can be used to rotate a point around a Z Axis (local or absolute).

If multiplied by a vector, the matrix will rotate that vector around the absolute Z-Axis. If this matrix is multiplied by another matrix, it will produce a new transformation matrix which, when multiplied by a vector, will rotate that vector around the transformation matrix's local z axis.
