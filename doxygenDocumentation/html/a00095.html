Detailed Description

A canvas that does zooming and 'ctrlScrolling'.

ctrlScrolling scrolls the canvas when the user holds the ctrl key and moves the mouse. To make this work: The OnPaint routine of any child class must call PrepareDC at the beginning to make this canvas draw properly. The OnMouseMove routine must be called when the mouse moves, so any child class must recieve mouse events.
