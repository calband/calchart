Detailed Description

The main frame of the CalChart program.

When you open CalChart, without opening a show file, you are greeted with an empty frame/window which you can use to open the files that you wish to view/edit. That is the "top frame," and it functions as the main frame for the CalChart program. If you close it, it closes all other windows, and you exit the program.
