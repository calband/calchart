Detailed Description

A four-dimensional vector.

This is used often for two-dimensional manipulations by zeroing the z component and maintaining the default 1.0 value for the w component. Matrices (which, in CalChart, always have 4x4 dimensions) are composed of four of these vectors, and those matrices can be used for a large number of applications, like rotating, scaling, translating, etc.
